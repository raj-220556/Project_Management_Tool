# Project Management Tool

## Problem Statement

Managing software development tasks, sprints, and team collaboration often requires complex tools. SudoTask provides a streamlined, developer-centric "Mini Jira" to track tasks, manage sprints, and integrate directly with development activity.

## Objectives

- Provide a Kanban-style dashboard (To-Do, In-Progress, Done).
- Enable Sprint-based task grouping and user assignments.
- Track project activity logs for transparency.
- (Advanced) Link development progress via simulated or real GitHub hooks.

## Key Features

- **User Authentication:** Secure login for Leads and Developers.
- **Sprint Management:** Create and manage time-bound sprints.
- **Task Board:** CRUD operations on tasks with status transitions.
- **Activity Feed:** A "who-did-what" timeline for the project.
- **Search/Filter:** Find tasks by assignee or priority.


## Technologies

- **Frontend:** HTML5, Tailwind CSS, JavaScript (React/Vanilla).
- **Backend:** PHP (using MVC pattern).
- **Database:** MongoDB (for flexible task schemas) or Firebase.
- **Version Control:** Git/GitHub.
